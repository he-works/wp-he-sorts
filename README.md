# HE SORTS

> WordPress 관리자 메뉴를 자유롭게 정렬·편집할 수 있는 플러그인

![Version](https://img.shields.io/badge/version-1.0.0-blue) ![WordPress](https://img.shields.io/badge/WordPress-5.0%2B-21759b) ![PHP](https://img.shields.io/badge/PHP-7.4%2B-777bb4) ![License](https://img.shields.io/badge/license-GPL--2.0-green)

---

## 소개

**HE SORTS**는 WordPress 관리자 대시보드의 왼쪽 메뉴를 드래그 앤 드롭으로 자유롭게 재배치할 수 있는 플러그인입니다.  
기본 메뉴 순서가 마음에 들지 않거나, 자주 쓰는 메뉴를 상단에 올리고 싶을 때 간편하게 사용할 수 있습니다.

---

## 주요 기능

- **드래그 앤 드롭 정렬** — 1뎁스(상단 메뉴)부터 3뎁스(중첩 하위 메뉴)까지 자유롭게 순서 변경
- **뎁스 이동** — 항목을 하위로 내리거나(들여쓰기), 상위로 올릴 수(내보내기) 있음
- **메뉴 이름 변경** — 각 항목의 표시 이름을 원하는 대로 수정
- **항목 숨기기** — 필요 없는 메뉴를 삭제하지 않고 숨김 처리
- **커스텀 메뉴 추가** — URL, 아이콘, 권한, 링크 대상을 지정하여 새 메뉴 항목 추가
- **GitHub 자동 업데이트** — WordPress 플러그인 업데이트 창에서 바로 업데이트 가능

## 설치 방법

### 직접 업로드
1. 이 저장소를 ZIP으로 다운로드합니다.
2. WordPress 관리자 → **플러그인 → 새로 추가 → 플러그인 업로드**에서 ZIP 파일을 선택합니다.
3. 설치 후 **활성화**합니다.

### FTP / 파일매니저
1. `plugin-HE-SORTS` 폴더를 `/wp-content/plugins/` 경로에 업로드합니다.
2. WordPress 관리자 → **플러그인**에서 **HE SORTS**를 활성화합니다.

---

## 사용 방법

1. 플러그인 활성화 후 관리자 메뉴 맨 하단의 **HE SORTS** 항목을 클릭합니다.
2. 또는 **플러그인 목록**에서 HE SORTS 옆의 **설정** 링크를 클릭합니다.

### 순서 변경
- 항목 왼쪽의 `⠿` 핸들을 드래그하여 원하는 위치로 이동합니다.

### 뎁스 조정
| 버튼 | 기능 |
|---|---|
| `→` (들여쓰기) | 앞 항목의 하위 메뉴로 이동 (최대 3뎁스) |
| `←` (내보내기) | 한 단계 상위 레벨로 이동 |

### 이름 변경
1. 변경할 항목을 클릭합니다.
2. 우측 **속성 패널**에서 표시 이름을 수정합니다.
3. **적용** 버튼을 누른 후 **저장**합니다.

### 항목 숨기기
- 항목 우측의 👁 버튼을 클릭하면 메뉴에서 숨겨집니다.
- 다시 클릭하면 표시됩니다.

### 커스텀 메뉴 추가
1. 상단 **항목 추가** 버튼을 클릭합니다.
2. 이름, URL, 아이콘, 권한, 링크 대상을 입력합니다.
3. **추가** 버튼을 누르면 트리에 추가됩니다.
4. **저장**합니다.

### 초기화
- **초기화** 버튼을 누르면 모든 설정이 삭제되고 WordPress 기본 메뉴 순서로 복원됩니다.
